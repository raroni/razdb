razdb
=====

The most advanced and most elegant DB system ever written in the history of man kind.

How to compile
--------------

`make`

How to use
----------

Creating a database:
`./razdb my_db.test c`

Registering a person:
`./razdb my_db.test s 1 Rasmus email_address@example.com`

Finding a person:
`./razdb my_db.test g 1`
